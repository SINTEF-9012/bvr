Starting yEd:
 1. Double-clock yed.jar

Loading pallettes:
 1. Right click on any pallette on the right side.
 2. Select "Import Section..."
 3. Select "CVL VSpec Tree.graphml"
 4. Repeat 1-3 and select "CVL Resolution.graphml"

Loading VSpec Tree:
 1. File->Open...
 2. Select graphml file

Automatic layout of VSpec tree:
 1. "Tools"->"Fit Node to Label..."
 2. "Layout"->"Tree..."
	2a) set "Port Style" to "Border Centered".
	2b) set "Minimum layer distance" to "20".
